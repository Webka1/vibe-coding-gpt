# Настройка базы данных Supabase

## 1. Создание таблиц

Перейдите в [Supabase Dashboard](https://supabase.com/dashboard) → ваш проект → SQL Editor.

Скопируйте содержимое файла `supabase-setup.sql` и выполните его.

## 2. Проверка таблиц

После выполнения SQL убедитесь, что созданы таблицы:
- `chats` - хранит информацию о чатах
- `messages` - хранит сообщения чатов

## 3. Структура таблиц

### Таблица `chats`
- `id` - уникальный идентификатор чата
- `user_id` - ID пользователя (из Supabase Auth)
- `title` - заголовок чата
- `created_at` - дата создания
- `updated_at` - дата последнего обновления

### Таблица `messages`
- `id` - уникальный идентификатор сообщения
- `chat_id` - ссылка на чат
- `role` - роль отправителя ('user' или 'assistant')
- `content` - текст сообщения
- `created_at` - дата создания

## 4. Безопасность

Настроены политики Row Level Security (RLS):
- Пользователи могут видеть только свои чаты и сообщения
- Пользователи могут создавать/редактировать только свои данные

## 5. Запуск приложения

После настройки базы данных запустите приложение:

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`.
